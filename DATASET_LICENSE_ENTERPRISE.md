- Technical support and data format documentation as specified in the agreement.

## How to get access

1. Read the dataset overview: [dataset/ENTERPRISE_DATASET.md](dataset/ENTERPRISE_DATASET.md).
2. Use the contact or request process described there (e.g. [dataset/scripts/request_enterprise_access.py](dataset/scripts/request_enterprise_access.py) or the linked form/email).
3. Sign a commercial license agreement with the data provider.
4. Receive access credentials and download instructions (e.g. S3/CDN or dedicated pipeline).
